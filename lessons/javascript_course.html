1.1 Setup Environment

Before we start coding, let's set up your JavaScript development environment. This is the foundation for all your JavaScript programming work.

What You'll Learn

  • Setting up a code editor
  • Using browser developer tools
  • Running your first JavaScript program
  • Understanding the JavaScript runtime

Step 1: Choose a Code Editor

We recommend using VS Code, Sublime Text, or WebStorm for the best development experience.

// Download VS Code from: https://code.visualstudio.com/
// Install JavaScript extensions for better development

Step 2: Browser Developer Tools

Every modern browser has built-in developer tools that include a JavaScript console.

// Open Developer Tools:
// Chrome/Edge: F12 or Ctrl+Shift+I
// Firefox: F12 or Ctrl+Shift+I
// Safari: Cmd+Option+I

Step 3: Create Your First HTML File

JavaScript runs in the browser, so we need an HTML file to include our JavaScript code.

<!DOCTYPE html>
<html>
<head>
  <title>My First JavaScript</title>
</head>
<body>
  <script>
    console.log("Hello, JavaScript!");
  </script>
</body>
</html>

Practice Exercise

Create the HTML file above and open it in your browser. Then open the developer tools and check the console to see your first JavaScript output!

1.2 Console & Variables

Learn the basics of outputting text and storing data in variables - the building blocks of JavaScript programming.

Console Output

The console.log() function is used to output text to the browser's console.

console.log("Hello, World!");
Hello, World!

console.log("Welcome to JavaScript!");
Welcome to JavaScript!

Variables

Variables are containers for storing data values. JavaScript has three ways to declare variables.

// Using let (recommended for variables that change)
let name = "Alice";
let age = 25;
let height = 1.75;

// Using const (for values that don't change)
const PI = 3.14159;
const MAX_SIZE = 100;

// Using var (older way, avoid in modern code)
var oldWay = "not recommended";

Data Types

JavaScript has several basic data types:

// String
let message = "Hello";
let singleQuote = 'Single quotes work too';

// Number
let integer = 42;
let decimal = 3.14;

// Boolean
let isTrue = true;
let isFalse = false;

// Undefined
let notDefined;

// Null
let empty = null;

Template Literals

Template literals (using backticks) allow you to embed expressions in strings.

let name = "Alice";
let age = 25;

console.log(`Hello, my name is ${name} and I am ${age} years old.`);
Hello, my name is Alice and I am 25 years old.

Practice Exercise

Create variables for your name, age, and favorite color. Then use template literals to print a sentence about yourself.

1.3 If-Else

Learn how to make decisions in your code using conditional statements. This is one of the most fundamental concepts in programming.

What You'll Learn

  • Basic if statements and conditions
  • If-else and if-else if-else structures
  • Comparison operators and logical operators
  • Ternary operators
  • Switch statements

Understanding Conditions

Conditions are expressions that evaluate to either true or false. JavaScript uses these boolean values to decide which code to execute.

// Simple conditions
let x = 5;
let y = 10;

console.log(x > y);
false

console.log(x < y);
true

console.log(x === 5);
true

Comparison Operators

JavaScript provides several operators to compare values:

// Comparison operators
let a = 10;
let b = 5;

console.log(`a > b: ${a > b}`);
a > b: true

console.log(`a < b: ${a < b}`);
a < b: false

console.log(`a === b: ${a === b}`);
a === b: false

console.log(`a !== b: ${a !== b}`);
a !== b: true

console.log(`a >= b: ${a >= b}`);
a >= b: true

console.log(`a <= b: ${a <= b}`);
a <= b: false

Basic If Statement

The if statement allows you to execute code only when a condition is true.

let age = 18;

if (age >= 18) {
  console.log("You are an adult");
  console.log("You can vote!");
}

You are an adult
You can vote!

If-Else Statement

Use else to provide an alternative when the condition is false.

let temperature = 25;

if (temperature > 30) {
  console.log("It's hot!");
  console.log("Stay hydrated!");
} else {
  console.log("It's comfortable");
  console.log("Enjoy the weather!");
}

It's comfortable
Enjoy the weather!

If-Else If-Else Statement

Use else if to check multiple conditions in sequence.

let score = 85;

if (score >= 90) {
  grade = "A";
} else if (score >= 80) {
  grade = "B";
} else if (score >= 70) {
  grade = "C";
} else if (score >= 60) {
  grade = "D";
} else {
  grade = "F";
}

console.log(`Your grade is: ${grade}`);

Your grade is: B

Logical Operators

Combine multiple conditions using logical operators: && (AND), || (OR), and ! (NOT).

let age = 25;
let hasLicense = true;
let hasCar = true;

// Using AND operator
if (age >= 18 && hasLicense) {
  console.log("You can drive!");
}

// Using OR operator
if (hasCar || hasLicense) {
  console.log("You have transportation options");
}

// Using NOT operator
if (!hasLicense) {
  console.log("You need to get a license first");
}

You can drive!
You have transportation options

Ternary Operator

The ternary operator provides a shorthand way to write simple if-else statements.

let age = 20;
let status = age >= 18 ? "adult" : "minor";
console.log(`You are an ${status}`);

You are an adult

Switch Statement

Use switch statements when you have many conditions to check against a single value.

let day = "Monday";

switch (day) {
  case "Monday":
    console.log("Start of the work week");
    break;
  case "Friday":
    console.log("TGIF!");
    break;
  case "Saturday":
  case "Sunday":
    console.log("Weekend!");
    break;
  default:
    console.log("Regular work day");
}

Start of the work week

Practice Exercise 1: Grade Calculator

Create a program that takes a student's score (0-100) and assigns a letter grade using if-else statements.

Practice Exercise 2: Password Validator

Write a program that checks if a password meets these criteria:

  • At least 8 characters long
  • Contains at least one uppercase letter
  • Contains at least one lowercase letter
  • Contains at least one number

Key Takeaways

  • Use if for single conditions
  • Use if-else for binary decisions
  • Use if-else if-else for multiple conditions
  • Combine conditions with &&, ||, !
  • Use ternary operator for simple conditions
  • Use switch statements for multiple value comparisons
  • Always use === for strict equality

1.4 Loops

Learn how to repeat code using loops - a fundamental concept in programming that allows you to execute code multiple times.

What You'll Learn

  • For loops and while loops
  • Loop control with break and continue
  • Iterating over arrays and objects
  • Nested loops
  • Common loop patterns

For Loop

The for loop is used to repeat code a specific number of times.

// Basic for loop
for (let i = 0; i < 5; i++) {
  console.log(`Count: ${i}`);
}

Count: 0
Count: 1
Count: 2
Count: 3
Count: 4

For...of Loop

The for...of loop is used to iterate over iterable objects like arrays.

// Loop through an array
let fruits = ["apple", "banana", "orange"];
for (let fruit of fruits) {
  console.log(fruit);
}

apple
banana
orange

For...in Loop

The for...in loop is used to iterate over object properties.

// Loop through object properties
let person = {
  name: "Alice",
  age: 25,
  city: "New York"
};

for (let key in person) {
  console.log(`${key}: ${person[key]}`);
}

name: Alice
age: 25
city: New York

While Loop

The while loop repeats code as long as a condition is true.

let count = 1;
while (count <= 5) {
  console.log(`Count: ${count}`);
  count++;
}

Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

Do...While Loop

The do...while loop executes the code block at least once, then checks the condition.

let number = 1;
do {
  console.log(`Number: ${number}`);
  number++;
} while (number <= 3);

Number: 1
Number: 2
Number: 3

Break and Continue

Use break to exit a loop early and continue to skip the current iteration.

// Using break
for (let i = 1; i <= 10; i++) {
  if (i === 6) {
    break;
  }
  console.log(i);
}

1
2
3
4
5
// Using continue
for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    continue;
  }
  console.log(i);
}

1
2
4
5

Nested Loops

You can nest loops inside other loops to create complex patterns.

// Creating a multiplication table
for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 3; j++) {
    console.log(`${i} x ${j} = ${i * j}`);
  }
}

1 x 1 = 1
1 x 2 = 2
1 x 3 = 3
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
3 x 1 = 3
3 x 2 = 6
3 x 3 = 9

Practice Exercise 1: Number Pattern

Create a program that prints the first 10 even numbers using a for loop.

Practice Exercise 2: Sum Calculator

Write a program that calculates the sum of numbers from 1 to 100 using a loop.

Practice Exercise 3: Array Processing

Create an array of numbers and use a loop to find the largest and smallest numbers in the array.

Key Takeaways

  • Use for loops when you know the number of iterations
  • Use while loops when you don't know the number of iterations
  • Use for...of to iterate over arrays
  • Use for...in to iterate over object properties
  • Use break to exit loops early
  • Use continue to skip iterations
  • Be careful with infinite loops!

2.1 Functions

Functions are reusable blocks of code that perform a specific task. They help organize code and prevent repetition.

What You'll Learn

  • Function declarations and expressions
  • Parameters and return values
  • Arrow functions
  • Scope and closures
  • Common function patterns

Function Declaration

Use the function keyword to declare a function.

function greet() {
  console.log("Hello, World!");
}

greet();

Hello, World!

Parameters and Arguments

Functions can accept parameters to work with dynamic data.

function add(a, b) {
  console.log(a + b);
}

add(5, 3);

8

Return Values

Use return to output a value from a function.

function multiply(x, y) {
  return x * y;
}

let result = multiply(4, 5);
console.log(result);

20

Arrow Functions

Arrow functions provide a concise syntax for writing functions.

const square = (num) => num * num;

console.log(square(7));

49

Practice Exercise

Create a function that calculates the area of a circle given its radius (use Math.PI).

Key Takeaways

  • Functions help reuse code
  • Use parameters for input
  • Return values for output
  • Arrow functions are concise

2.2 Arrays

Arrays are ordered collections of values that allow you to store multiple items in a single variable.

What You'll Learn

  • Creating and accessing arrays
  • Array methods
  • Iterating through arrays
  • Multi-dimensional arrays

Creating Arrays

Use square brackets to create an array.

let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]);

apple

Array Methods

Common methods include push, pop, shift, unshift.

let numbers = [1, 2, 3];
numbers.push(4);
console.log(numbers);

[1, 2, 3, 4]

Higher-Order Methods

Use map, filter, reduce for array transformations.

let nums = [1, 2, 3, 4];
let doubled = nums.map(n => n * 2);
console.log(doubled);

[2, 4, 6, 8]

Practice Exercise

Create an array of numbers and use filter to get only even numbers.

Key Takeaways

  • Arrays are zero-indexed
  • Use methods to manipulate
  • Higher-order functions are powerful

2.3 Objects

Objects are collections of key-value pairs that represent real-world entities.

What You'll Learn

  • Creating objects
  • Accessing properties
  • Methods in objects
  • Object destructuring

Creating Objects

Use curly braces to create objects.

let person = { name: "Alice", age: 25 };
console.log(person.name);

Alice

Methods

Functions inside objects are called methods.

let calculator = {
  add: function(a, b) { return a + b; }
};
console.log(calculator.add(2, 3));

5

Destructuring

Extract values from objects easily.

let { name, age } = person;
console.log(name, age);

Alice 25

Practice Exercise

Create an object representing a book with properties and a method to get summary.

Key Takeaways

  • Objects store related data
  • Access via dot or bracket notation
  • Methods add behavior

2.4 DOM Basics

The Document Object Model (DOM) represents the structure of HTML documents.

What You'll Learn

  • Selecting elements
  • Manipulating content
  • Changing attributes
  • Creating elements

Selecting Elements

Use methods like getElementById, querySelector.

let element = document.getElementById("myId");
let elements = document.querySelectorAll(".class");

Manipulating Content

Change text and HTML content.

element.textContent = "New text";
element.innerHTML = "<p>New HTML</p>";

Creating Elements

Dynamically add elements to the page.

let newDiv = document.createElement("div");
document.body.appendChild(newDiv);

Practice Exercise

Create a button that changes the text of a paragraph when clicked.

Key Takeaways

  • DOM allows JavaScript to interact with HTML
  • Select elements to manipulate
  • Change content and attributes dynamically
