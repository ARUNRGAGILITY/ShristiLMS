1.3 If-Else Logic

Learn how to make decisions in your code using conditional statements. This is one of the most fundamental concepts in programming that allows your code to respond differently based on various conditions.

What You'll Learn

  • Basic if statements and conditions
  • If-else and if-elif-else structures
  • Comparison operators and logical operators
  • Nested conditions and complex logic
  • Real-world applications of conditional logic

Understanding Conditions

Conditions are expressions that evaluate to either True or False. Python uses these boolean values to decide which code to execute.

# Simple conditions
x = 5
y = 10

print(x > y)
False

print(x < y)
True

print(x == 5)
True

Comparison Operators

Python provides several operators to compare values:

# Comparison operators
a = 10
b = 5

print(f"a > b: {a > b}")
a > b: True

print(f"a < b: {a < b}")
a < b: False

print(f"a == b: {a == b}")
a == b: False

print(f"a != b: {a != b}")
a != b: True

print(f"a >= b: {a >= b}")
a >= b: True

print(f"a <= b: {a <= b}")
a <= b: False

Basic If Statement

The if statement allows you to execute code only when a condition is true. The code block must be indented (typically 4 spaces).

age = 18

if age >= 18:
    print("You are an adult")
    print("You can vote!")

You are an adult
You can vote!

If-Else Statement

Use else to provide an alternative when the condition is false. This ensures that some code always executes.

temperature = 25

if temperature > 30:
    print("It's hot!")
    print("Stay hydrated!")
else:
    print("It's comfortable")
    print("Enjoy the weather!")

It's comfortable
Enjoy the weather!

If-Elif-Else Statement

Use elif (else if) to check multiple conditions in sequence. This is more efficient than multiple separate if statements.

score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"Your grade is: {grade}")

Your grade is: B

Logical Operators

Combine multiple conditions using logical operators: and, or, and not.

age = 25
has_license = True
has_car = True

# Using AND operator
if age >= 18 and has_license:
    print("You can drive!")

# Using OR operator
if has_car or has_license:
    print("You have transportation options")

# Using NOT operator
if not has_license:
    print("You need to get a license first")

You can drive!
You have transportation options

Nested Conditions

You can nest if statements inside other if statements to create complex decision trees.

weather = "sunny"
temperature = 75

if weather == "sunny":
    if temperature > 80:
        print("Go to the beach!")
    elif temperature > 60:
        print("Go for a walk in the park")
    else:
        print("Stay inside, it's chilly")
else:
    print("Check the weather forecast")

Go for a walk in the park

Checking for Membership

Use in and not in to check if a value exists in a sequence (list, tuple, string).

fruits = ["apple", "banana", "orange"]
favorite = "banana"

if favorite in fruits:
    print(f"{favorite} is available!")
else:
    print(f"{favorite} is not available")

name = "Alice"
if "a" in name.lower():
    print("The name contains 'a'")

banana is available!
The name contains 'a'

2.1 Functions

Functions are reusable blocks of code that perform specific tasks. They help organize and modularize your code.

Defining Functions

Use the def keyword to create a function.

def greet(name):
    print(f"Hello, {name}!")

greet("Alice")

Hello, Alice!

Functions with Parameters

Functions can accept multiple parameters and have default values.

def greet_with_title(name, title="Mr."):
    print(f"Hello, {title} {name}!")

greet_with_title("Smith")
Hello, Mr. Smith!

greet_with_title("Johnson", "Dr.")
Hello, Dr. Johnson!

Functions with Return Values

Functions can return values using the return statement.

def add_numbers(a, b):
    return a + b

result = add_numbers(5, 3)
print(result)

8

2.2 Lists & Tuples

Lists and tuples are ordered collections that can store multiple items. Lists are mutable, while tuples are immutable.

Lists

Lists are created using square brackets and can be modified after creation.

# Creating a list
fruits = ["apple", "banana", "orange"]
print(fruits)
['apple', 'banana', 'orange']

# Adding items
fruits.append("grape")
print(fruits)
['apple', 'banana', 'orange', 'grape']

# Accessing items
print(fruits[0])
apple

Tuples

Tuples are created using parentheses and cannot be modified after creation.

# Creating a tuple
coordinates = (10, 20)
print(coordinates)
(10, 20)

# Accessing tuple elements
x, y = coordinates
print(f"X: {x}, Y: {y}")
X: 10, Y: 20

2.3 Dictionaries

Dictionaries store key-value pairs and are perfect for organizing related data.

Creating Dictionaries

Dictionaries use curly braces and store data as key-value pairs.

# Creating a dictionary
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

print(person)
{'name': 'Alice', 'age': 25, 'city': 'New York'}

Accessing Dictionary Values

You can access values using keys and modify dictionaries.

# Accessing values
print(person["name"])
Alice

# Adding new key-value pair
person["email"] = "alice@email.com"
print(person)
{'name': 'Alice', 'age': 25, 'city': 'New York', 'email': 'alice@email.com'}

2.4 Sets

Sets are unordered collections of unique elements, perfect for removing duplicates and set operations.

Creating Sets

Sets are created using curly braces or the set() function.

# Creating a set
fruits = {"apple", "banana", "orange", "apple"}
print(fruits)
{'apple', 'banana', 'orange'}

# Note: duplicates are automatically removed
numbers = set([1, 2, 2, 3, 3, 4])
print(numbers)
{1, 2, 3, 4}

Set Operations

Sets support mathematical set operations like union, intersection, and difference.

set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

# Union
union = set1 | set2
print(union)
{1, 2, 3, 4, 5, 6}

# Intersection
intersection = set1 & set2
print(intersection)
{3, 4}

3.1 File Handling

Learn how to read from and write to files, a crucial skill for data processing and persistence.

Reading Files

Use the open() function with the 'r' mode to read files.

# Reading a file
with open("sample.txt", "r") as file:
    content = file.read()
    print(content)

# Reading line by line
with open("sample.txt", "r") as file:
    for line in file:
        print(line.strip())

Writing Files

Use the 'w' mode to write files. This will overwrite existing content.

# Writing to a file
with open("output.txt", "w") as file:
    file.write("Hello, World!\n")
    file.write("This is a test file.")

# Appending to a file
with open("output.txt", "a") as file:
    file.write("\nThis line was appended.")

3.2 Error Handling

Learn how to handle errors gracefully using try-except blocks to make your programs more robust.

Basic Try-Except

Use try-except blocks to catch and handle exceptions.

try:
    number = int(input("Enter a number: "))
    result = 10 / number
    print(f"Result: {result}")
except ValueError:
    print("Please enter a valid number!")
except ZeroDivisionError:
    print("Cannot divide by zero!")

Finally Block

The finally block always executes, regardless of whether an exception occurred.

try:
    file = open("data.txt", "r")
    content = file.read()
except FileNotFoundError:
    print("File not found!")
finally:
    file.close()
    print("File closed successfully.")

Learn how to organize code into modules and use external packages to extend Python's functionality.

Creating Modules

Modules are Python files that can be imported and used in other programs.

# math_utils.py

def multiply(a, b):

PI = 3.14159

# main.py
import math_utils

Using Built-in Modules

Python comes with many built-in modules like math, random, and datetime.

import math
import random
from datetime import datetime

# Using math module
print(math.sqrt(16))
4.0

7

2024-01-15 14:30:25.123456

Creating Classes

Classes are blueprints for creating objects with attributes and methods.

class Car:
    def __init__(self, brand, model, year):
        self.brand = brand
        self.model = model
        self.year = year

    def display_info(self):
        print(f"{self.year} {self.brand} {self.model}")

my_car = Car("Toyota", "Camry", 2020)
my_car.display_info()
2020 Toyota Camry

Inheritance

Inheritance allows classes to inherit attributes and methods from parent classes.

class ElectricCar(Car):
    def __init__(self, brand, model, year, battery_capacity):
        super().__init__(brand, model, year)
        self.battery_capacity = battery_capacity

    def display_info(self):
        super().display_info()
        print(f"Battery: {self.battery_capacity} kWh")

tesla = ElectricCar("Tesla", "Model 3", 2023, 75)
tesla.display_info()
2023 Tesla Model 3
Battery: 75 kWh
